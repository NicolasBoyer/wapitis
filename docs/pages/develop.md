# Développer

Pendant le développement, il est recommandé d'utiliser :

```bash
npx wapitis dev
```

ou

```bash
npx wapitis electron --dev
```

permettant de mettre à jour et de visualiser ses modifications à la volée grâce au compileur et au watcher intégrés.
